# ACCIDENT-AND-INSURANCE-PROJECT

## Project Description
This project analyzes road accident and insurance data . It includes data preprocessing, exploratory data analysis (EDA), visualizations.

## Objectives
- Analyze accident patterns.
- Explore relationships between accident and insurance data.
- Visualize accident hotspots.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook
